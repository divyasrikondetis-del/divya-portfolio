# 🌐 Divya Portfolio

A personal portfolio website built using JavaScript to showcase my skills, projects, achievements, and professional journey as a developer.

## 🚀 Live Demo

https://divya-portfolio-nu.vercel.app

## ✨ Features

* Responsive design
* Modern user interface
* Projects showcase
* Skills section
* About Me section
* Contact information

## 🛠️ Technology Used

* JavaScript

## 📂 Project Structure

```text
divya-portfolio/
│
├── src/
├── index.html

├── package-lock.json
└── configuration files
```

## ▶️ Run Locally

```bash
git clone https://github.com/divyasrikondetis-del/divya-portfolio.git
cd divya-portfolio
npm install
npm run dev
```
